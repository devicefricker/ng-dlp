# ng-dlp
Downloader for Newgrounds (This time it **will** happen)

# Usage
### General usage
```[ng-dlp-win.exe OR ./ng-dlp-linux] [url]```
## Arguments
### General arguments
```
  --path, -P   Output path, relative to executable path
  
  --test, -t   Run test suite and exit (requires data dir)
  
  --version, -V   Print version and exit
```
### Movie extraction arguments
```
  --quality, -q		Download video in specified quality (valid inputs are '1080p', '720p', or '360p'.)
```
### Metadata arguments
```  
  --writeinfojson	Write JSON containing metadata about submission
```
