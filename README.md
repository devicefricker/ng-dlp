# ng-dlp
Downloader for Newgrounds

# Usage
### General usage
```[ng-dlp-win.exe OR ./ng-dlp-linux] [url]```
## Arguments
### General arguments
```
  --path, -P   Output path, relative to executable path
  
  --test, -t   Run test suite and exit (requires data dir)
  
  --version, -V   Print version and exit
```
